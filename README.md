## Deskripsi Project
Project ini adalah backend sederhana berbasis **Node.js + Express** yang menggunakan **whatsapp-web.js** untuk menghubungkan aplikasi web Anda dengan WhatsApp Web.

Fitur utama:

- Generate QR untuk login WhatsApp  
- Cek status koneksi WhatsApp  
- Mendapatkan informasi profil WhatsApp (nama & nomor)  
- Logout / reset session WhatsApp  
- API berjalan lokal agar dapat diakses aplikasi Laravel
